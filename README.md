# Dummy Certification Authority
A quick and dirty way for creating a Root Certification Authority (CA) and one Intermediate CA for testing.

## Build
```
git clone https://github.com/kintoandar/dummy_ca.git
cd dummy_ca
./build.sh
```
Yep, that easy!
